# 用户手册

## 第一章 系统概述

欢迎使用企业资源规划系统（ERP系统）。本系统是企业级管理软件，集成了采购、销售、库存、生产、财务等核心业务模块，帮助企业实现精细化管理。

## 第二章 系统登录

### 2.1 登录地址
访问地址：https://erp.company.com

### 2.2 登录步骤
1. 打开浏览器，输入登录地址
2. 输入用户名和密码
3. 点击"登录"按钮
4. 首次登录需修改初始密码

### 2.3 忘记密码
1. 点击登录页"忘记密码"链接
2. 输入注册手机号或邮箱
3. 获取验证码并重置密码
4. 使用新密码重新登录

## 第三章 主界面介绍

登录成功后进入系统主界面，主要包含：
- **顶部导航栏**：系统Logo、搜索框、消息中心、用户头像
- **左侧菜单栏**：功能模块菜单，支持折叠展开
- **主工作区**：显示具体业务内容和操作界面
- **底部状态栏**：当前时间、在线人数、系统版本

## 第四章 核心功能操作

### 4.1 采购管理

#### 新建采购订单
1. 点击"采购管理" → "采购订单"
2. 点击"新建"按钮
3. 选择供应商
4. 添加采购商品（商品编码、数量、单价）
5. 确认订单信息
6. 点击"提交审批"

#### 采购入库
1. 点击"采购管理" → "入库单"
2. 点击"新建入库单"
3. 选择关联的采购订单
4. 确认入库商品和数量
5. 选择入库仓库
6. 提交审核

### 4.2 销售管理

#### 创建销售订单
1. 点击"销售管理" → "销售订单"
2. 点击"新建"
3. 选择客户
4. 添加销售商品
5. 确认价格和数量
6. 提交审批

#### 销售出库
1. 点击"销售管理" → "出库单"
2. 新建出库单
3. 关联销售订单
4. 确认出库商品和仓库
5. 打印出库单

### 4.3 库存管理

#### 库存查询
路径：库存管理 → 库存查询

支持以下查询方式：
- 按商品编码查询
- 按仓库查询
- 按批次查询
- 组合条件查询

#### 库存盘点
1. 点击"库存管理" → "盘点单"
2. 选择盘点仓库
3. 系统自动生成库存明细
4. 录入实际盘点数量
5. 系统自动计算差异
6. 提交盘点报告

### 4.4 财务管理

#### 费用报销
1. 点击"财务管理" → "费用报销"
2. 选择费用类型（差旅费、招待费、交通费等）
3. 填写费用明细
4. 上传发票附件
5. 选择审批人
6. 提交报销申请

## 第五章 常见问题

Q: 为什么我的菜单看不到某些功能？
A: 这可能是因为您的账号没有相应权限，请联系管理员开通。

Q: 如何修改个人登录密码？
A: 点击右上角用户头像 → "账号设置" → "修改密码"

Q: 单据审批流程是什么？
A: 提交单据后，系统会自动通知审批人，您可以在"待我审批"中查看待办事项。
