# 新闻报道

## 北京2024年3月15日电

### 我国成功发射全球首颗AI大模型卫星

今日上午10时，我国在酒泉卫星发射中心使用长征二号F运载火箭，成功将"盘古-1"AI卫星送入预定轨道。这颗卫星是全球首颗搭载千亿参数AI大模型的卫星，标志着我国在太空AI领域取得重大突破。

**卫星性能参数**

| 参数 | 指标 |
|------|------|
| 卫星重量 | 3.2吨 |
| 轨道高度 | 550公里（太阳同步轨道） |
| AI算力 | 1000 PFLOPS |
| 模型参数量 | 1000亿 |
| 设计寿命 | 8年 |

**四大核心能力**

这颗卫星具备以下四大核心能力：
1. 实时对地观测与AI分析：能够在轨完成遥感图像的智能解译，目标识别准确率达95%以上
2. 边缘计算与数据处理：支持在轨AI推理，减少数据传输量90%
3. 太空实验平台：为科研机构提供太空AI实验环境
4. 应急通讯支持：在地面通讯中断时提供应急数据中继

**战略意义**

国防科工局相关负责人表示，"盘古-1"卫星的成功发射，将极大提升我国太空感知和智能化能力。在气象预报、环境监测、灾害预警等领域具有广阔的应用前景。

"这颗卫星相当于给地球装上了一颗'太空大脑'。"项目总师王建平在接受采访时表示，"它能够自主识别需要重点关注的区域，大大提高数据获取和处理的效率。"

**民用应用前景**

在民用领域，"盘古-1"卫星将服务于：
- 精准农业：作物病虫害早期预警
- 城市管理：交通流量实时分析
- 海洋监测：非法捕捞行为识别
- 环境保护：生态变化动态监测

**国际反响**

此次发射在国际社会引起广泛关注。多国航天机构表示祝贺，并表达了合作意愿。美国航天局发言人在例行记者会上表示，中国在太空AI领域的进展"令人印象深刻"。

## 上海3月14日电

### 首届全球AI治理大会在上海开幕

首届全球AI治理大会今日在上海国际会议中心开幕，来自120个国家和地区的代表参会。大会聚焦AI伦理、AI安全、AI与就业等热点议题。

大会发表了《上海AI治理宣言》，呼吁各国加强AI领域的对话与合作，共同应对AI带来的挑战。

## 广州3月13日电

### 我国新能源汽车保有量突破1亿辆

据公安部统计，截至今年2月底，我国新能源汽车保有量已达1.02亿辆，占汽车总量的30%以上。这一数据表明，我国新能源汽车发展进入新阶段。

专家表示，新能源汽车的普及将加速推进碳中和目标的实现。预计到2030年，我国新能源汽车保有量将突破3亿辆。
