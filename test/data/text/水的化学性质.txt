水的化学性质

水（化学式：H2O）是生命之源，是地球上最常见的化合物之一，也是人体的重要组成部分。

一、水的基本性质

1. 物理性质
   - 外观：无色、无味、无臭的透明液体
   - 沸点：100°C（1个大气压）
   - 冰点：0°C
   - 密度：4°C时最大，为1g/cm³
   - 比热容：4.18 J/(g·°C)
   - 表面张力：72.8 mN/m（20°C）

2. 化学性质
   - 水是极性分子
   - 能溶解多种物质（溶剂）
   - 可参与化学反应

二、水的化学反应

1. 水的合成
   2H₂ + O₂ → 2H₂O（燃烧反应）

2. 水的分解
   2H₂O → 2H₂ + O₂（电解）

3. 水合反应
   CuSO₄ + 5H₂O → CuSO₄·5H₂O（生成五水合硫酸铜）

4. 水解反应
   NaCl + H₂O → NaOH + HCl（盐的水解）

三、水的特殊性质

1. 水的异常膨胀
   - 4°C时密度最大
   - 冰的密度小于水，所以冰浮在水面上
   - 对水生生物生存至关重要

2. 高比热容
   - 调节气候和体温
   - 海洋性气候的形成原因

3. 高表面张力
   - 毛细现象
   - 浸润与非浸润

四、水的分类
1. 纯水：几乎不含杂质
2. 矿泉水：含有矿物质
3. 硬水：含较多Ca²⁺、Mg²⁺
4. 软水：不含或含较少Ca²⁺、Mg²⁺
