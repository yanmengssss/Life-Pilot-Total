## HTTP vs HTTPS 对比

| 维度 | HTTP | HTTPS |
|------|------|-------|
| **全称** | HyperText Transfer Protocol | HTTP Secure |
| **传输方式** | 明文传输 | 加密传输 |
| **端口** | 80 | 443 |
| **证书** | 无需证书 | 需要SSL/TLS证书 |
| **安全性** | 低 | 高 |
| **SEO影响** | 无加分 | Google给予排名加分 |
| **加载速度** | 稍快 | 略慢（加密开销） |

## HTTPS 的加密机制

### SSL/TLS 握手过程
1. 客户端向服务器请求连接
2. 服务器发送SSL证书
3. 客户端验证证书有效性
4. 双方协商加密算法
5. 建立加密通道

## 为什么现代网站都用HTTPS？
1. **数据安全**：防止窃听、篡改
2. **身份验证**：确认网站真实性
3. **浏览器要求**：Chrome等浏览器标记HTTP为不安全
4. **合规要求**：PCI-DSS等安全标准要求
5. **SEO优化**：搜索引擎排名因素之一

## 证书类型
- **DV证书**：域名验证，仅加密
- **OV证书**：组织验证，更高信任度
- **EV证书**：扩展验证，最高级别（绿色地址栏）
