## 概念关系

```
人工智能（AI）
    └── 机器学习（ML）
            └── 深度学习（DL）
```

深度学习是机器学习的一个子集，两者存在包含关系。

## 核心区别

| 维度 | 机器学习 | 深度学习 |
|------|----------|----------|
| 特征提取 | 人工手动提取特征 | 自动从数据中学习特征 |
| 算法复杂度 | 相对简单 | 复杂，多层神经网络 |
| 数据需求 | 较少 | 大量数据 |
| 计算资源 | 普通设备即可 | 需要GPU等高性能计算 |
| 可解释性 | 较好 | 较差（黑箱问题） |
| 训练时间 | 较短 | 较长 |

## 技术实现对比

### 机器学习
```python
# 传统机器学习流程
1. 人工提取特征（人工设计）
2. 选择算法（决策树/SVM/随机森林等）
3. 训练模型
4. 预测输出
```

### 深度学习
```python
# 深度学习流程
1. 原始数据输入
2. 多层神经网络自动学习特征（端到端）
3. 直接输出结果
```

## 典型应用场景

| 场景 | 机器学习 | 深度学习 |
|------|----------|----------|
| 图像分类 | SVM + 人工特征 | CNN卷积神经网络 |
| 语音识别 | GMM-HMM模型 | RNN/LSTM/Transformer |
| NLP | 传统方法 | BERT/GPT等预训练模型 |

## 各自优势

**机器学习优势：**
- 数据量小时也能表现良好
- 特征可解释性强
- 计算成本低

**深度学习优势：**
- 复杂问题表现优异
- 特征自动学习
- 泛化能力强
