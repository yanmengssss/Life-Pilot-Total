# LifePilot — AI 日程与任务助理（毕业设计）

## 项目简介

LifePilot 是一款以大语言模型（LLM）和 LangGraph 多工作流 Agent 为核心的**全栈 AI 智能任务助理系统**，支持自然语言对话、智能任务规划、私人知识库问答、语音交互及出行搜索，帮助用户将复杂的日程管理与资料检索一体化解决。

项目由 **8 个独立子系统**协同构成，全部容器化部署，已上线生产环境（[lifepilot.website](https://lifepilot.website)）。

## 核心亮点

### 1. LangGraph 多工作流 Agent 架构

设计并实现了基于 LangGraph 的**意图路由 → 多工作流**架构，顶层 Router 节点（DeepSeek）识别用户意图，自动分发至 5 条专属工作流：

- **`build_todo`**：三角色协作（执行者 × 检查者 × 格式化器） + **Human-in-the-Loop**（`interrupt()` 中断等待用户确认，确认后通过 MCP 协议批量写入 MySQL）
- **`what_to_do`**：任务查询与分析（执行者 × 检查者 × 管家三角色，纯自然语言回复）
- **`trivel` / `howToGo`**：**ReAct Agent** 架构，LLM 自主调用高德地图 MCP + 小红书 MCP 完成多跳出行规划
- **`rag`**：Weaviate 向量检索 + GPT-4o 兜底的知识库问答

所有工作流通过 **SSE（Server-Sent Events）** 以 `think / response / confirm / refresh / done` 等事件类型实时推送给前端。

### 2. RAG 私人知识库（ai-server / FastAPI）

独立 Python FastAPI 服务，提供 RAG 全链路能力：
- 支持 PDF / Word / Excel / PPT / 图片（OCR）/ TXT 多格式文档解析
- 文本分块 → 向量化 → 写入 Weaviate，按 `appName`（Index）+ `userID` 双重数据隔离
- 同时提供 **TTS 流式语音合成**、**ASR 语音转文字**（SiliconFlow）、**七牛云 OSS** 管理接口

### 3. MCP 工具微服务（LifePilot_mcp）

基于 **Model Context Protocol（MCP）** 构建独立工具微服务，通过 Streamable HTTP Transport 对外暴露 5 个预定义工具（`create_task / get_tasks / update_task / delete_task / create_tag`）。AI Agent 只能调用规范工具，无法执行任意 SQL，数据库操作权限收口于 MCP 服务，显著提升安全性。

### 4. 异步任务调度与邮件提醒

基于 **Redis Zset + Node.js 定时轮询**实现高性能任务调度：
- `zrangebyscore` 拉取到期任务 → `zrem` 原子抢占（防并发重复处理）
- 从 MySQL 读取任务详情，判定状态后通过 Nodemailer 发送 HTML 任务超时/提醒邮件
- 邮件发送后按 `tipsFrequency` 重新计算下次触发时间，`zadd` 回队，实现循环提醒

### 5. 日志监控与数据管理平台（Metaphorical & OmniBase）

自研两个独立平台：

- **Metaphorical**（[metaphorical.yanmengsss.xyz](https://metaphorical.yanmengsss.xyz)）：集中式日志看板，接收 `yanmengs-logs` SDK（已发布正式 npm 包 `^1.0.0`）上报的结构化日志，按 `projectKey` 隔离存储至 MongoDB，提供分页查询与多维筛选可视化界面

- **OmniBase**（[omnibase.yanmengsss.xyz](https://omnibase.yanmengsss.xyz)）：多数据源可视化管理平台，通过 Next.js API Routes（BFF 层）直连 Weaviate / MySQL / MongoDB / Redis，提供 Schema 管理、数据 CRUD、SQL 编辑器、多标签页会话隔离等功能

### 6. 共享 SDK 封装与 npm 发布

封装并正式发布两个 npm 包：
- **`yanmengs-rag-package`**（`^1.0.1`）：统一 RAG / TTS / ASR / 七牛 OSS HTTP 客户端，前后端共用，隐藏 ai-server 地址与鉴权 Key
- **`yanmengs-logs`**（`^1.0.0`）：轻量日志上报 SDK，任意项目引入即可通过 `projectKey + tableName` 上报日志至 Metaphorical

### 7. 全链路 Docker + Jenkins CI/CD 部署

6 个子项目均配置独立 `Jenkinsfile`，Push 触发 Pipeline → 构建 Docker 镜像 → 推送 Registry → 远程服务器拉取重启容器 → Nginx 反向代理绑定域名，实现一键自动化部署。
